生成和配置SSH秘钥

  • 打开’Git Bash’创建SSH目录，可以先查看该目录是否存在，秘钥是否存在。（秘钥存在目录:C:\Users\Administrator\.ssh）
  • 创建目录:mkdir ~/.ssh

  • cd ~/.ssh ,使用ssh-keygen -t rsa -C “自己的邮箱”，生成密钥

    没有设置密码直接敲两次回车，生成秘钥

  • 将SSH目录下的id_rsa.pub用记事本打开，复制，进入github设置界面添加SSH


SourceTree配置SSH

  • 进入选项界面，在SSH客户端配置里勾选OpenSSH选项会自动识别设备里SSH秘钥，如果不存在需要先创建秘钥，点击确定完成配置。
